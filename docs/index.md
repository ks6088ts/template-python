# template-python

A modern GitHub template for Python projects, with `uv`, Ruff, mypy, pytest,
multi-stage Docker, and a security-conscious CI/CD pipeline (CodeQL,
pip-audit, gitleaks, Trivy, Sigstore-signed releases).

See the [README](https://github.com/ks6088ts/template-python#readme) for
the full feature list and quick-start instructions.
